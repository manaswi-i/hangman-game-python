# Hangman Game (Python, Pygame)

An interactive Hangman game built using Python and Pygame.

## Features
- Graphical interface using Pygame
- Random word selection
- User input-based guessing
- Win/Loss conditions with attempt tracking

## Tech Stack
- Python
- Pygame

## How to Run
1. Install pygame:
   pip install pygame

2. Run the file:
   python main.py

## Purpose
This project was created to strengthen understanding of game logic, user interaction, and graphical rendering using Python.
